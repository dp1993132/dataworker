# DataWorker

```
Yottachain 项目使用的数据采集工具
```

Install

```
## 安装
go get github.com/yuin/gluamapper
go get github.com/yuin/gopher-lua
go get github.com/dp1993132/dataworker
cd `GOPATH`/github.com/dp1993132/dataworker
make build
```

## Usage
```bash
# 在out目录下能找到对应的执行文件

dataworker ./example/test.lua

```
